# Changelog

## 0.1.0 — 2026-07-21

- Initial FHIR R4 formalization of the interoperability and data-quality methodology.
- Added 35 canonical profiles, 11 extensions, and 9 source logical models.
- Added terminology artifacts for six data-quality metrics and mapping/validation metadata.
- Added 11 ConceptMaps and 12 StructureMaps with readable FML sources.
- Added CapabilityStatement, two OperationDefinitions, two SearchParameters, examples, and canonical exchange Bundle.
- Added deterministic integrity checks, offline R4 validation, artifact inventory, traceability matrix, and FHIR NPM package assembly.
